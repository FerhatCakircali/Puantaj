# iOS Test - Mac/iPhone Olmadan

## 🎯 Ne Yapacaksın:

### 1. GitHub'a Push Yap
```bash
git add .
git commit -m "iOS test"
git push
```

### 2. GitHub Actions'da Bekle (10 dakika)
- GitHub.com > Projen > **Actions** sekmesi
- **iOS Build** otomatik başlayacak
- Bekle

### 3. Build'i İndir
- Build bitince **Artifacts** > **ios-runner-app** indir
- ZIP dosyası gelecek

### 4. Appetize.io'da Test Et
- https://appetize.io > **Try Demo**
- **Upload** > ZIP'i yükle
- iPhone 14 Pro seç
- Test et!

Evet. Ayda 100 dakika ücretsiz.
